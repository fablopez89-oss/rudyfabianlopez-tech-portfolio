# Bash Scripts Portfolio

Colección de scripts Bash útiles para administración de sistemas — listos para usar como ejemplo en un portafolio de GitHub.

**Archivos incluidos**
- check_server_status.sh — Verifica conectividad (ping) y servicios via systemctl.
- resource_check.sh — Muestra uso de CPU, memoria y disco.
- update_system.sh — Detecta gestor de paquetes y actualiza el sistema.
- backup_disks.sh — Ejemplo de backup usando rsync o tar.
- rotate_logs.sh — Rotación simple de logs por tamaño/antigüedad.
- user_audit.sh — Informe de usuarios, sudoers y últimas conexiones.
- monitor_network.sh — Estadísticas básicas de red y conexiones.
- deploy_example.sh — Ejemplo simple de despliegue desde git y reinicio de servicio.

**Licencia**
Usa estos scripts en tus proyectos. Asegúrate de revisarlos antes de ejecutarlos en producción.

**Notas**
- Muchos scripts requieren privilegios (sudo/root).
- Ajusta rutas, exclusiones y parámetros a tu infraestructura.
- Son plantillas educativas; testéalas en entornos controlados antes de usarlas en servidores críticos.
