# Windows, Azure y Microsoft 365 Automation Portfolio

Scripts profesionales en PowerShell para automatizar tareas comunes en entornos híbridos, Windows Server, Azure AD y Microsoft 365.

## Contenido
- Creación masiva de usuarios (AD On-Prem y Azure AD)
- Eliminación masiva de usuarios
- Asignación masiva de licencias M365
- Auditoría de privilegios
- Monitoreo de servidores Windows
- Verificación de recursos y aplicaciones
- Reportes automáticos
- Ejecución programada (Task Scheduler)
- Ejemplos de automatización para Azure Automation
